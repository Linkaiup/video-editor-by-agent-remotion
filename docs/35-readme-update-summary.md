- 复杂度评分系统

- API 成本降低 80%

- 网络层重试（3次）

- 每步产出具名制品文件
- 便于调试和增量恢复

#### 4️⃣ 自动设计系统
- 智能提取设计 Tokens
- 自动生成设计规范

#### 5️⃣ 三重验证系统
- Lint + TypeCheck + 视觉快照
- 质量评分（0-100）

#### 6️⃣ Sub-agent 隔离策略
- 每个 beat 独立生成
- 避免上下文污染

#### 7️⃣ 完整的可观测性
- 结构化日志
- 性能指标
- 实时进度

#### 8️⃣ 灵活的配置系统
- 支持 OpenAI 官方 API
